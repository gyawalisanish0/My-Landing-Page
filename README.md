# Personal landing page

Sanish Gyawali's personal site — bio, featured projects, and a support section.

Built with Vite + React + TypeScript, no UI framework.

## Development

```
npm install
npm run dev
```

## Build

```
npm run build
```

Content lives in `src/data/content.ts` — update it to change bio, project, or link details.
Donate links in `src/components/Donate.tsx` are placeholders (`href="#"`) until real accounts are wired up.
